LLHelper
========

## How to Use

### Install requirements
```sh
pip install -r requirements.txt
```

### Run on Production Environment
```sh
FLASK_APP=app.py FLASK_ENV=devlopment python2 -m flask run
```

### Run on Production Environment
```sh
FLASK_APP=app.py FLASK_RUN_PORT={PORT} python2 -m flask run
```
